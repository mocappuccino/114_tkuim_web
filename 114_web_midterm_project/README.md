### 專案簡介
- 使用者可根據自己持有的食材及需求，搜尋有興趣的食譜，並可詳細查看該料理所需食材數量及大致步驟。
- 使用者也可分享自創食譜，並上傳到網站上。

### 使用技術
- HTML5、CSS3
- Bootstrap 5（樣式與佈局）
- 原生 JavaScript（DOM 操作、事件、localStorage）

### 功能特色
- 食材勾選篩選：使用者勾選食材後會即時自動篩選配方。
- 篩選模式：支援「包含 (include)」與「完全符合 (haveAll)」兩種檢索語意。
- 匹配高亮與排序：在結果中高亮顯示配方中被選取到的必要食材，並依符合比例進行排序。
- 食譜詳情：點選結果可查看完整材料、步驟與份量換算（輸入份量會動態調整數量）。
- 分享表單：表單驗證、偵測填寫錯誤與授權勾選。
